Utopia: A Twitter Clone

For CSC 322, I collaborated with my team on the "Utopia" project, a comprehensive web application for our software engineering class. We used Flask for the backend with a SQLAlchemy database and a React frontend. We implemented features like reading and managing posts/comments, user subscriptions, job postings, and messaging systems. This hands-on experience enhanced our skills in system design, team collaboration, and applying software engineering principles.

Carbon Route

A project developed for the DevFest 2024 Hackathon. We aim to provide users with an accurate calculation of how much their ordered amazon products produces along its path to your home. We utilized Google Map's Matrix API to convert inputted location data into longitude and latitude, which would be fed into the Carbon emission API alongside an inputted package weight to calculate carbon emission by pound.

Secure Chatroom

For my Computer Security class, I was tasked with implementing Triple Diffie Helman Encryption on a chatroom program. This process entails creation of ephemeral keys on both clients' sides, which through several transformations allow messages to be shared without explicit sharing of these keys.
